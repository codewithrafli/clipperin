Clipperin: Turn long videos into shorts. Locally.

No cloud. No subscription. No watermark. A modular video clipping solution that runs entirely on your hardware.

Get Started Documentation

The Problem

Modern video tools have become service-wrapped traps. Most "AI" clippers require you to upload your files to their servers, pay monthly fees to remove watermarks, and lock essential features behind a paywall.

Recurring monthly subscriptions
Data privacy concerns (cloud uploads)
Mandatory branding/watermarks

The Difference

Clipperin is a tool, not a service. It provides a modular architecture with clear separation between core logic, CLI, and optional web UI. It respects your machine and your autonomy.

One-time setup, free forever
Zero data leaves your computer
Total control over the pipeline

The Pipeline

A transparent look at what happens during a single execution run.

01

Download

Source material is pulled using yt-dlp or processed from local storage.

02

Transcribe

The audio stream is passed through Whisper model for timestamped text generation.

03

Analyze

AI or rule-based detection finds chapters and highlights in the transcript.

04

Select

Review detected chapters and select which segments to convert into clips.

05

Render

Export via FFmpeg to 9:16 vertical format with burned-in subtitles.

AI Transcription

Local Whisper model (no API key needed) with support for 99 languages.

Vertical Ready

Native support for 9:16, 1:1, 4:5 formats with smart cropping and subtitle styles.

Modular Architecture

Use as Python library, CLI tool, or web UI - all the same core under the hood.

AI Provider Options

Choose Gemini (FREE), Groq, OpenAI, or rule-based for chapter detection.

CLI First

Power-user CLI with optional web UI for visual chapter review and selection.

Zero Tracking

No telemetry, no phone-home, no analytics. Your data stays local.

# Install CLI
$ pip install -e "./packages/clipperin-cli[full]"

# Run full pipeline
$ clipperin pipeline "https://youtube.com/watch?v=..." -o ./output

[INFO] Downloading video...
[INFO] Transcribing with Whisper...
[INFO] Analyzing content...
[INFO] Found 8 chapters
[DONE] Chapters saved to ./output/chapters.json

# Render clips
$ clipperin render video.mp4 chapters.json -o ./clips
[DONE] Exported 8 clips to ./clips/

Open Source Philosophy

Clipperin is built with clear separation of concerns: clipperin-core (library), clipperin-cli (interface), clipperin-ui (optional web). This means you can use and extend it however you want - as a Python library in your own projects, as a CLI tool for automation, or with the web UI for visual workflows.

Modular. Extensible. Yours.
